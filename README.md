Word cloud
=======
A simple word cloud generator for python beginner by using the wordcloud package.

Prerequisites knowledge
------
* Basic list slicing
* Basic function
* File I/O
* Exception handling
* Module imports

Prerequisites setting
--------
* Wordcloud module's setting
* Matplotlib module's setting
